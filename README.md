# 🚀 Chef GH  

يقدم المشروع وصفات الطبخ وتعليم الطبخ للمبتدئين والمبتدئات في عالم الطبخ، كما يعرض منصة للتعريف عن التخصصات المهنية للأشخاص وخبراتهم.  

## 📌 الميزات  
- 📖 **وصفات متنوعة**: عرض وصفات طبخ مفصلة مع المكونات والخطوات.  
- 🎥 **دروس فيديو**: محتوى مرئي لتعلم الطبخ بأسلوب سهل.  
- 👨‍🍳 **ملفات تعريف للطهاة**: يمكن للمستخدمين عرض مهاراتهم وخبراتهم في الطهي.  
- 🔍 **بحث متقدم**: إمكانية البحث عن وصفات بناءً على المكونات، الوقت، والمستوى.  
- 📲 **تصميم متجاوب**: يعمل على الهواتف الذكية والأجهزة اللوحية بسلاسة.
